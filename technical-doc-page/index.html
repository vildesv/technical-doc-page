Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

It supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Python is widely used in web development, data analysis, AI, and more.

Its syntax allows developers to express concepts in fewer lines of code.

Python is open-source and has a large community contributing to its ecosystem.

print("Hello, world!") # This is a comment in Python
  • Easy to learn
  • Open-source
  • Rich standard library
  • Cross-platform
  • Community support
Variables and Types

Variables in Python do not require explicit declaration to reserve memory space.

The declaration happens automatically when you assign a value.

Python is dynamically typed, meaning variables can change type.

Common data types include int, float, str, list, tuple, dict, and bool.

Use type() to check the type of a variable.

x = 5 name = "Alice"
  • int
  • str
  • float
  • bool
  • list
Control Flow

Python uses if, elif, and else for conditional logic.

Loops are used to repeat a block of code.

for loops iterate over sequences, while while loops execute as long as a condition is true.

Python uses indentation instead of braces to define blocks.

Use break and continue to control loop behavior.

if x > 0: for i in range(5):

  • elif
  • else
  • for
  • while
Functions

Functions are defined using the def keyword in Python.

You can pass arguments and return values.

Functions help make code modular and reusable.

Default arguments and keyword arguments are supported.

Anonymous functions can be created with lambda.

def greet(name): return "Hello " + name
  • def
  • return
  • arguments
  • lambda
  • modularity
Modules and Packages

Modules are Python files with reusable code.

Packages are directories containing multiple modules and an __init__.py file.

Use import to include modules in your code.

You can create your own modules or use built-in ones like math or os.

Third-party packages can be installed via pip.

import math from datetime import datetime
  • import
  • from ... import ...
  • pip
  • __init__.py
  • reuse code
